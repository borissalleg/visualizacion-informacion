## Objetivos generales del curso

Al finalizar el curso, el estudiante estará en capacidad de:

1. Comprender los fundamentos teóricos y perceptuales que sustentan una
   visualización de información efectiva.
2. Seleccionar el tipo de visualización adecuado según la naturaleza de los datos y
   el objetivo comunicativo.
3. Construir cuadros de mando (dashboards) estáticos e interactivos usando
   herramientas de código abierto y comerciales.
4. Utilizar las principales librerías de visualización en Python (Matplotlib,
   Seaborn, Plotly, Bokeh, Folium) y compararlas con alternativas como D3.js y
   ggplot2.
5. Construir geovisualizaciones (mapas coropléticos, mapas de densidad,
   interpolación espacial) para el análisis territorial de datos.

## Requisitos previos

- Conocimientos básicos de Python (variables, funciones, estructuras de datos).
- Familiaridad con `pandas` para manipulación de datos tabulares.
- Nociones básicas de estadística descriptiva.

## Cómo ejecutar los ejemplos

Todos los ejemplos de código de este sitio son autocontenibles y usan datasets de
ejemplo (incluidos en las librerías o generados sintéticamente), por lo que no
requieren fuentes de datos externas. Consulta el `README.md` del repositorio para
instrucciones de instalación y ejecución.

!!! example "Ejercicio propuesto transversal"
    Antes de comenzar, identifica una visualización que hayas visto recientemente
    (en prensa, redes sociales o un reporte de trabajo) y analízala críticamente:
    ¿qué tipo de dato representa?, ¿qué canal visual usa (posición, color,
    tamaño)?, ¿es engañosa o efectiva? Guarda tus notas: las retomaremos en la
    Unidad 1.
